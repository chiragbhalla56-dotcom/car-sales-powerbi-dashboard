 Car Sales Power BI Dashboard

 Project Overview

This project is an interactive **Power BI dashboard** created to analyze car sales data and extract meaningful business insights.
The dashboard focuses on sales performance, customer behavior, deal size contribution, and year-wise trends using KPIs and visuals.

---

 Key KPIs

* Total Sales
* Total Orders
* Total Quantity Sold
* Average Order Value

---

 Dashboard Features

* Sales distribution by **Product Line**
* Year-wise **sales trend analysis**
* Deal size contribution (**Small, Medium, Large**)
* Customer-wise sales and order count
* Interactive slicers for:

  * Year
  * Product Line
  * Deal Size
  * Territory

---

Tools & Technologies Used

* Microsoft Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)

---

 Key Business Insights

* Classic Cars generate the highest overall sales revenue.
* Medium and Large deal sizes contribute the majority of total sales.
* Sales show consistent growth across the analyzed years.
* A small number of customers contribute significantly to total revenue.
